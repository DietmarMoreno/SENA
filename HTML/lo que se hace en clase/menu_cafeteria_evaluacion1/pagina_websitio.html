explicacion de la pagina

para la creacion de la pagina se usaron las distintas etiquetas las cuales fueron proporcionadas por el instructor, las que ayudaron para ordenar la pagina y colocar su contenido, fue algo medio dificil entender bien como hacer que ciertas lineas se ajustaran, pero fue cuestion de tiempo para poder ordenar todo dentro de la misma, para el contenido de la pagina como los nombres y precios toco usar un tabla y con ayuda del css fue posible hacer que queden separados y no juntos como es costumbre, para la parte del link de color fue gracias a un comando llamado (a:hover) para que cambie de color a la hora de colocar el cursor encima del mismo enlace, para las letras se uso (font-family) y se uso la fuente (calibri) para el contenido y los titulos (impact)
